# nclex-practice-quiz

## What is it?
A simple **and terrifying** single-file web app containing 120 NCLEX style practice questions which target exam 1 content for Northeasterns NRSG-3320 Adult 1 Medical-Surgical Nursing course.

## What makes it different?
Through good old fashion positive punishment you will fear getting the answer wrong deep in your soul

## Tips
Volume all the way up for full experience
